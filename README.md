# Agentic AI Social Media Manager

## Overview
An AI-powered social media automation system built using n8n.

## Features
- Generates marketing content using Gemini AI
- Creates social media post images
- Uploads media to Cloudinary
- Publishes posts to Instagram automatically
- Scheduled posting workflow

## Tech Stack
- n8n
- Gemini AI
- Cloudinary
- Instagram Graph API
- Puppeteer

## Workflow
1. Trigger on schedule
2. Generate content
3. Create image
4. Upload image
5. Generate caption
6. Publish to Instagram
